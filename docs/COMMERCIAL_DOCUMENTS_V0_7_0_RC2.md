# Коммерческие документы v0.7.0rc2

Реализованы детерминированные расчёты КП и счёта, версии документов, PDF тестового шаблона, audit trail и mock-обновление бюджета amoCRM.

Формулы:
- цена продажи = закупочная цена × (1 + наценка / 100);
- строка = количество × цена продажи;
- база = subtotal − скидка + доставка;
- НДС рассчитывается программно с округлением HALF_UP до 0.01;
- итог хранится в БД, PDF не является источником истины.

Ограничения:
- фирменный шаблон заказчика не предоставлен;
- live amoCRM не проверен;
- mock budget не вызывает внешний API.

Исправление RC2:
- smoke-тест явно освобождает SQLAlchemy engine до удаления временной SQLite БД на Windows.
