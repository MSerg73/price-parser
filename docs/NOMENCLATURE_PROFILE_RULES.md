# Правила профилей номенклатуры

Версия: 1.1  
Действует с Price Parser `0.7.1rc4`.

## RULE-PROFILE-001 — КРУГ и ПРУТОК

Статус: **подтверждено пользователем 22.06.2026**.

Для целей поиска `КРУГ` и `ПРУТОК` являются одной группой круглого проката.

### Нормализация результата

Исходные значения профиля сохраняются:

- строка поставщика с обозначением `КРУГ` остаётся `КРУГ`;
- строка поставщика с обозначением `ПРУТОК` остаётся `ПРУТОК`;
- автоматическое переписывание основной таблицы не выполняется.

Это нужно для трассируемости и проверки результата по исходному файлу.

### Размеры

Для обоих профилей:

- `Размер 1` — диаметр;
- `Размер 2` — длина, если она явно указана в исходной строке;
- дополнительные размеры сохраняются только при наличии в исходной строке;
- отсутствующие значения не выдумываются.

### Поиск

Запрос с профилем `ПРУТОК` рассматривает:

- `ПРУТОК`;
- `КРУГ`.

Запрос с профилем `КРУГ` рассматривает:

- `КРУГ`;
- `ПРУТОК`.

При одинаковой марке и одинаковом диаметре обе позиции считаются точными
совпадениями. Профиль, буквально указанный в запросе, показывается первым.
Затем показывается второй профиль из той же поисковой группы.

### Ограничения

- `ТРУБА`, `ЛЕНТА`, `ЛИСТ`, `ПЛИТА` и другие профили в группу не входят;
- эквивалентность действует только для поиска и размерной логики;
- исходный профиль поставщика не теряется;
- правило не разрешает автоматически заменять товар в коммерческом предложении
  без проверки менеджером.
