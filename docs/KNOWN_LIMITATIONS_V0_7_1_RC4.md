# Известные ограничения — 0.7.1rc4

1. Программа имеет CLI, GUI отсутствует.
2. Неизвестные структуры обрабатываются generic fallback; спорные строки могут
   быть направлены в REVIEW.
3. Исходные клиентские прайсы не входят в update/delivery ZIP.
4. Неизвестный прайс заказчика проверяется онлайн или полной видеозаписью.
5. LLM не применяется автоматически к основному XLSX.
6. Тестовое задание не включает amoCRM, почту, сервер, production-базу и PDF.
7. Новое количество результатов поиска `пруток БрБ2 ф20` ещё должно быть
   зафиксировано локальным прогоном на трёх исходных прайсах.
8. Историческое значение `152/4` больше не является актуальным критерием.
9. Исходные значения `КРУГ` и `ПРУТОК` сохраняются в основной таблице, хотя для
   поиска они образуют одну группу.
