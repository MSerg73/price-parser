# Public anonymized repository

This export contains the application source code, automated tests, selected
technical documentation and synthetic fixtures.

Deliberately excluded:

- original supplier price lists;
- customer and supplier names;
- source file hashes and acceptance evidence;
- generated XLSX files;
- live LLM reports, replay payloads and audit data;
- local configuration, secrets and virtual environments;
- historical customer-specific release manifests.

Supplier-specific adapters are published under neutral identifiers:
Supplier Alpha, Supplier Beta, Supplier Gamma and Supplier Delta.
